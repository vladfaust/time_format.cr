    Installation

    Add this to your application's shard.yml:

    dependencies:
  time_format:
    github: vladfaust/time_format.cr
    version: ~> 0.1.1

    This shard follows Semantic Versioning 2.0.0, so see releases and change the version accordingly.

    require "time_format"
